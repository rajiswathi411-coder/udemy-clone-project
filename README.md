# Udemy Clone

**Udemy Clone** is a **front-end project** replicating the look and feel of the Udemy online learning platform. Built using **HTML and CSS**, this project showcases responsive design, clean layout, and user-friendly interface.

## Features

* Responsive course listing pages
* Navigation bar with dropdown menus
* Attractive course cards with images, titles, and prices
* Footer with essential links and contact info
* Fully built with HTML and CSS (no backend)

## Tech Stack

* **HTML5**
* **CSS3**
